一、数据库表结构 (server/src/storage/database/shared/schema.ts)
================================================================================

-- users 用户表
CREATE TABLE users (
  id SERIAL PRIMARY KEY,
  username VARCHAR(50) UNIQUE NOT NULL,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- user_progress 用户进度表
CREATE TABLE user_progress (
  id SERIAL PRIMARY KEY,
  user_id INTEGER REFERENCES users(id) UNIQUE,
  points INTEGER DEFAULT 0,
  mastered_words INTEGER DEFAULT 0,
  updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- points_history 积分历史表
CREATE TABLE points_history (
  id SERIAL PRIMARY KEY,
  user_id INTEGER REFERENCES users(id),
  type VARCHAR(10) NOT NULL CHECK (type IN ('earn', 'spend')),
  amount INTEGER NOT NULL,
  reason TEXT,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

================================================================================
二、后端 API 接口
================================================================================

【养成系统 API】前缀: /api/growth

1. POST /api/growth/register    注册用户
   请求: { username: string }
   响应: { code, msg, data: { id, username, points, masteredWords, level } }

2. GET  /api/growth/users       获取所有用户列表
   响应: { code, msg, data: UserInfo[] }

3. GET  /api/growth/user/:id    获取单个用户信息
   响应: { code, msg, data: UserInfo }

4. POST /api/growth/earn        获取积分（掌握单词）
   请求: { userId: number, word: string }
   响应: { code, msg, data: { points, masteredWords, level } }

5. POST /api/growth/spend       消耗积分（兑换）
   请求: { userId: number, amount: number, reason: string }
   响应: { code, msg, data: { points, masteredWords } }

6. GET  /api/growth/history/:userId  积分历史明细
   响应: { code, msg, data: PointsRecord[] }

7. GET  /api/growth/stats/:userId    积分统计
   响应: { code, msg, data: { totalEarned, totalSpent, balance } }

【听写 API】前缀: /api/dictation

1. POST /api/dictation/upload-image          上传图片到TOS
2. POST /api/dictation/recognize-all-words    LLM识别图片中所有单词
3. POST /api/dictation/speak-word-both       获取单词美式+英式发音URL
4. POST /api/dictation/upload-audio          上传录音
5. POST /api/dictation/recognize-handwriting  手写识别
6. POST /api/dictation/recognize-speech       语音识别
7. POST /api/dictation/check-answer          检查拼写答案
8. POST /api/dictation/get-new-words         获取新单词词库
9. POST /api/dictation/get-review-words      获取复习词库
10. POST /api/dictation/add-to-review        加入复习词库
11. POST /api/dictation/remove-from-review   从复习词库移除
12. POST /api/dictation/add-word             手动添加单词
13. POST /api/dictation/remove-word          手动删除单词

================================================================================
三、养成系统核心逻辑 (server/src/growth/growth.service.ts)
================================================================================

等级计算规则：
  stars = masteredWords % 10        (每10星 = 1月亮)
  moons  = floor(masteredWords / 10) % 10  (每10月 = 1太阳)
  suns   = floor(masteredWords / 100)

示例：
  掌握 0 个单词  → 0☀️ 0🌙 0⭐ (空星)
  掌握 5 个单词  → 0☀️ 0🌙 5⭐
  掌握 15 个单词 → 0☀️ 1🌙 5⭐
  掌握 235 个单词 → 2☀️ 3🌙 5⭐

积分规则：
  - 每掌握一个单词 +1 积分
  - 积分可消耗（兑换），等级只升不降
  - 答对时自动调用 earn API 发放积分

四、前端页面说明
================================================================================

【首页】src/pages/index/index.tsx
  - 未选择用户：显示用户列表 + 注册入口
  - 已选择用户：显示等级图标、积分余额、功能入口（听写/拍照/词库/个人中心/积分管理）
  - 用户信息缓存到 Taro Storage key='current_user'

【个人中心】src/pages/profile/index.tsx
  - 显示用户头像、名称、注册日期
  - 等级展示（太阳/月亮/星星图标）
  - 积分概览（当前余额/累计获取/累计消耗）
  - 跳转积分管理入口

【积分管理】src/pages/points/index.tsx
  - 积分余额卡片（渐变背景）
  - 积分兑换表单（数量 + 原因）
  - 积分明细列表（earn/spend 历史记录，带时间戳）

【听写流程】src/pages/dictation/index.tsx
  - 答对时自动调用 /api/growth/earn 发放积分
  - 更新本地缓存的用户积分和等级信息
