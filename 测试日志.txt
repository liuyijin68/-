**测试结论**：✅ 通过


### 3. 发音接口测试

**接口**：POST /api/dictation/speak-word-both

**测试命令**：
```bash
curl -X POST http://localhost:3000/api/dictation/speak-word-both \
  -H "Content-Type: application/json" \
  -d '{"word":"put up"}'
```

**测试结果**：
```json
{
  "code": 200,
  "msg": "success (Google TTS fallback)",
  "data": {
    "usAudioUrl": "https://translate.google.com/translate_tts?ie=UTF-8&tl=en&client=tw-ob&q=put%20up",
    "ukAudioUrl": "https://translate.google.com/translate_tts?ie=UTF-8&tl=en-GB&client=tw-ob&q=put%20up"
  }
}
```

**测试结论**：✅ 通过（TTS SDK 配额不足时自动回退 Google TTS）

**额外测试**：
```bash
curl -X POST http://localhost:3000/api/dictation/speak-word-both \
  -H "Content-Type: application/json" \
  -d '{"word":"important"}'
```
结果：返回美式+英式发音 URL ✅

**测试结论**：✅ 通过

``

**断言验证**：
