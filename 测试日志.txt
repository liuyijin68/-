| `"put up 张贴"` | `"put up"` | ✅ `"put up"` |

1. 找到第一个 `/` 或中文字符位置

## 图片识别流程模拟

### LLM 返回 JSON 场景

**输入**：图片包含单词列表

**LLM 返回**：
```json
["put up", "important /ɪmˈpɔːtnt/ adj. 重要的", "notice /ˈnəʊtɪs/", 
 "locker", "drawer", "at the back (of) 在...后面", "corner", 
 "bookcase", "screen", "at school", "different from"]
```

**cleanWord 处理后**：
```
["put up", "important", "notice", "locker", "drawer", 
 "at the back (of)", "corner", "bookcase", "screen", 
 "at school", "different from"]
```

**结果**：11个纯英文单词/短语 ✅

### LLM 返回非 JSON 场景

**LLM 返回**：
```
put up 张贴
important /ɪmˈpɔːtnt/ adj. 重要的
notice n. 通知
```

**按行拆分 + cleanWord 处理后**：
```
["put up", "important", "notice"]
```

**结果**：3个纯英文单词 ✅
