### Bug 1: 修复图片识别无单词

**问题分析**：
- `Network.request` 返回 `{ statusCode, data }`，其中 `data` 是后端响应体
- 后端响应体结构为 `{ code: 200, msg: "success", data: { words: [...], count } }`
- 原代码直接用 `res?.data?.words` 取值，漏掉了中间层 `data`

**修复方案**：
```ts
// 修复前（错误）
const words = res?.data?.words  // undefined

// 修复后（正确）
const responseData = res?.data
const wordsData = responseData?.data?.words || responseData?.words
```

**修复位置**：
- `src/pages/upload/index.tsx` 第97行（handleRecognize函数）

**测试样本**：
样例图片应识别出：put up, important, notice, locker, drawer, at the back (of), corner, bookcase, screen, at school, different from（共11个单词/短语）

### Bug 2: 修复朗读无声

**问题分析**：
- 原代码使用 Web `new Audio()` API，小程序不支持
- TTS SDK 配额不足时无法生成音频

**修复方案**：
1. 改用 `Taro.createInnerAudioContext()` 播放音频（微信小程序原生API）
2. 后端添加 Google TTS 回退方案（免费、无需认证）

```ts
// 修复后：使用 Taro 音频 API
function playAudioUrl(url: string): Promise<void> {
  return new Promise((resolve, reject) => {
    const audioCtx = Taro.createInnerAudioContext()
    audioCtx.src = url
    audioCtx.autoplay = true
    audioCtx.onEnded(() => {
      audioCtx.destroy()
      resolve()
    })
    audioCtx.onError((err) => {
      audioCtx.destroy()
      reject(err)
    })
  })
}
```

**后端回退方案**：
```ts
// Google Translate TTS URL（免费）
const googleTtsUrl = (text: string, lang: string) =>
  `https://translate.google.com/translate_tts?ie=UTF-8&tl=${lang}&client=tw-ob&q=${encodeURIComponent(text)}`

// 美式发音：lang = 'en'
// 英式发音：lang = 'en-GB'
```

**修复位置**：
- `src/pages/dictation/index.tsx` 第25-48行（playAudioUrl函数）
- `server/src/dictation/dictation.controller.ts` 第198-274行（speakWordBoth方法）

### Bug 3: 修复录音未识别

**问题分析**：
- 原代码将录音数据直接发送给 ASR，但 ASR SDK 需要音频 URL
- 录音使用 Web Speech API，小程序不支持

**修复方案**：
1. 使用 `Taro.getRecorderManager()` 录音（微信小程序原生API）
2. 录音 → 上传到对象存储 → 获取 URL → 调用 ASR SDK

```ts
// Fix Bug 3: 录音 → 上传到存储 → ASR识别
const startRecording = useCallback(() => {
  const recorderManager = Taro.getRecorderManager()
  recorderManager.onStop((res) => {
    if (res.tempFilePath) {
      handleVoiceResult(res.tempFilePath)
    }
  })
  recorderManager.start({
    format: 'wav',
    sampleRate: 16000,
    numberOfChannels: 1,
  })
}, [])

// 录音后处理
const handleVoiceResult = async (tempFilePath: string) => {
  // Step 1: 上传录音文件到对象存储
  const uploadRes = await Network.uploadFile({
    url: '/api/dictation/upload-audio',
    filePath: tempFilePath,
    name: 'audio',
  })
  
  // Step 2: 调用 ASR 识别
  const res = await Network.request({
    url: '/api/dictation/recognize-speech',
    method: 'POST',
    data: { audioUrl },
  })
}
```

**修复位置**：
- `src/pages/dictation/index.tsx` 第154-241行
- `server/src/dictation/dictation.controller.ts` 第276-310行（upload-audio端点）
- `server/src/dictation/dictation.controller.ts` 第354-382行（recognize-speech端点）

### Bug 4: 修复中文含义判断误判

**问题分析**：
- 原比对逻辑缺少 `trim()`、`toLowerCase()`、标点符号去除
- 用户输入带空格或标点的答案被判定错误

**修复方案**：
```ts
// Fix Bug 4: trim whitespace, normalize, remove punctuation
const normalizedAnswer = userAnswer.trim().toLowerCase().replace(/[，,。.！!？?；;：:、\s]+/g, '')
const normalizedMeanings = correctMeanings.map(m =>
  m.trim().toLowerCase().replace(/[，,。.！!？?；;：:、\s]+/g, ''),
)

// 多含义匹配：任意一个正确即判定正确
const isCorrect = normalizedMeanings.some(meaning => {
  // 精确匹配
  if (normalizedAnswer === meaning) return true
  // 包含匹配（用户可能只说了部分含义）
  if (meaning.includes(normalizedAnswer) && normalizedAnswer.length >= 1) return true
  if (normalizedAnswer.includes(meaning) && meaning.length >= 1) return true
  return false
})
```

**修复位置**：
- `src/pages/dictation/index.tsx` 第145-151行（checkMeaning函数）
- `server/src/dictation/dictation.controller.ts` 第385-418行（check-answer端点）

### Bug 5: 新增手写识别

**问题分析**：
- 原代码缺少 Canvas 手写板输入英文功能

**修复方案**：
1. 使用 Taro Canvas 2D API 创建手写板
2. 导出 Canvas 图片 → 上传 → LLM 多模态识别手写英文

```ts
// Fix Bug 5: Canvas手写板 — 使用 Taro Canvas 2D API
const initCanvas = useCallback(() => {
  const query = Taro.createSelectorQuery()
  query.select('#handwritingCanvas').fields({ node: true, size: true }).exec((res) => {
    if (res && res[0] && res[0].node) {
      const canvas = res[0].node
      const ctx = canvas.getContext('2d')
      const dpr = Taro.getSystemInfoSync().pixelRatio
      canvas.width = res[0].width * dpr
      canvas.height = res[0].height * dpr
      ctx.scale(dpr, dpr)
      ctx.lineCap = 'round'
      ctx.lineJoin = 'round'
      ctx.lineWidth = 3
      ctx.strokeStyle = '#333'
      ctxRef.current = ctx
    }
  })
}, [showCanvas])

// 手写识别提交
const handleHandwritingSubmit = async () => {
  // 导出Canvas为图片
  const res = await Taro.canvasToTempFilePath({
    canvasId: 'handwritingCanvas',
    fileType: 'png',
  })
  // 上传并识别
  const recognizeRes = await Network.request({
    url: '/api/dictation/recognize-handwriting',
    method: 'POST',
    data: { imageUrl },
  })
}
```

**后端新增端点**：
```ts
@Post('recognize-handwriting')
async recognizeHandwriting(@Body() body: { imageUrl: string }) {
  const llmResponse = await this.llmClient.invoke([
    {
      role: 'user',
      content: [
        { type: 'image_url', image_url: { url: imageUrl } },
        { type: 'text', text: 'Recognize handwritten English text...' },
      ],
    },
  ], { model: 'doubao-seed-2-0-lite-260215' })
  
  return { code: 200, msg: 'success', data: { text: llmResponse.content.trim() } }
}
```
