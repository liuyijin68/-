一、修复目标
============================================

问题1：上传图片后点"识别单词"仍然提示"未识别到单词"
问题2：手动录入单词后，点"开始听写"，提示"词库数据解析失败"

根因分析：
1. **存储格式不一致**：词库管理页面用 `Taro.setStorageSync` 直接存数组，而其他页面用 `JSON.stringify` 存字符串，读取时又用 `JSON.parse`，导致解析报错。
2. **图片识别后单词列表为空**：LLM 调用失败、返回内容解析不匹配、或环境变量缺失。需要增强鲁棒性和错误提示。

### 修复1：统一存储格式（解决"词库数据解析失败"）

**涉及文件**：
- `src/pages/vocabulary/index.tsx`
- `src/pages/dictation/index.tsx`

**修复方案**：
添加统一存储格式辅助函数，兼容字符串和数组两种格式

```tsx
// 读取辅助函数
const loadFromStorage = (key: string): WordItem[] => {
  const raw = Taro.getStorageSync(key);
  if (!raw) return [];
  try {
    // 如果是字符串则 parse，否则直接当作数组（兼容旧数据）
    return typeof raw === 'string' ? JSON.parse(raw) : raw;
  } catch {
    return [];
  }
};

// 写入辅助函数
const saveToStorage = (key: string, data: WordItem[]) => {
  Taro.setStorageSync(key, JSON.stringify(data));
};
```

**修复位置**：
- vocabulary/index.tsx 第17-30行：添加 loadFromStorage 和 saveToStorage
- vocabulary/index.tsx 第46-49行：loadVocabularies 使用 loadFromStorage
- vocabulary/index.tsx 第53-61行：setCurrentWords 使用 saveToStorage
- dictation/index.tsx 第24-33行：添加 loadWordsFromStorage
- dictation/index.tsx 第98-109行：useEffect 中使用 loadWordsFromStorage + 空词库友好提示
- dictation/index.tsx 第317-330行：addToReview / removeFromReview 使用 loadWordsFromStorage

### 修复前（错误代码）

**vocabulary/index.tsx - 错误的存储读取**：
```tsx
const loadVocabularies = () => {
  const storedNewWords = Taro.getStorageSync('new_vocabulary') || [];
  // 问题：如果存储的是 JSON 字符串，|| [] 不会触发，导致后续解析错误
  setNewWords(storedNewWords);
};
```

**dictation/index.tsx - 错误的存储写入**：
```tsx
const addToReview = (word: WordItem) => {
  try {
    const stored = Taro.getStorageSync('review_vocabulary')
    const reviewList: WordItem[] = stored ? JSON.parse(stored) : []
    // 问题：如果 vocabulary 页面存的是数组，JSON.parse 会报错
    // ...
  } catch {
    // ignore - 隐藏了错误
  }
}
```

### 修复后（正确代码）

**vocabulary/index.tsx - 统一读取**：
```tsx
const loadVocabularies = () => {
  setNewWords(loadFromStorage('new_vocabulary'));
  setReviewWords(loadFromStorage('review_vocabulary'));
};
```

**dictation/index.tsx - 统一写入**：
```tsx
const addToReview = (word: WordItem) => {
  const reviewList = loadWordsFromStorage('review_vocabulary')
  // 兼容两种格式，不会报错
  if (!reviewList.some((w) => w.word === word.word)) {
    reviewList.push({ ...word, date: new Date().toISOString().split('T')[0] })
    Taro.setStorageSync('review_vocabulary', JSON.stringify(reviewList))
  }
}
```
